1. **找到激活导航地址**: 打开[激活网站](https://3.jetbra.in/){target="_blank"}，选择成功连接的地址。

2. **下载激活插件**: 进入网站后找到顶部点击下载最新版的激活插件压缩包

3. **运行插件**: 打开解压的目录

    === ":fontawesome-brands-windows: Windows"

        ``` bash title=""
        scripts/install-all-users.vbs
        ```

    === ":simple-apple: MacOS | :simple-linux: Linux"

        ``` bash title=""
        scripts/install.sh
        ```

4. **激活**: 打开要激活的产品，届时会提示你激活

    - 打开步骤2的网址，然后找到对应的工具或者插件名称，然后鼠标移到工具名称下面的 ***** 那里，然后点击复制即可

    - 返回到工具中，点击激活码，然后将复制的激活码粘贴进去点击激活即可，插件和主题激活同理

    - 激活完成
