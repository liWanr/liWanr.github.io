## 基本说明

本站尊重您的隐私并致力于保护您的个人数据，所以最终呈现的是静态网站，不会收集您的任何数据。但依赖的外部服务可能会有收集数据的行为，^^**本站已尽力选择合规、可信的第三方服务商，但对其数据处理行为不作直接担保**^^，建议您根据需求自行查阅这些服务商的隐私政策。

## 信息收集

### 1. 评论功能

本站使用 Giscus 服务提供评论功能，该服务本身不会收集任何信息。但其利用的 GitHub Discussions 可能会在您留言时收集以下信息：

- GitHub 账户信息
- IP 地址
- 浏览器信息
- 评论内容

Giscus 及 GitHub 均为第三方服务，都有其自身的隐私政策，建议您查看
[**Giscus 隐私政策**](https://github.com/giscus/giscus/blob/main/PRIVACY-POLICY.md) /
[**GitHub 通用隐私政策**](https://docs.github.com/en/site-policy/privacy-policies/github-general-privacy-statement)
以了解他们如何处理您的数据。

### 2. 订阅新闻简报

本站使用**站主自建的外部闭源订阅系统**管理新闻简报订阅，您的电子邮件地址仅用于接收新闻简报通知。您也可以随时点击任何简报邮件中的退订链接或直接与我联系以取消订阅。当您订阅本站的新闻简报时，订阅系统会收集您的:

- 电子邮件地址
- 订阅日期和时间
- IP 地址（用于防止欺诈）

其中所有数据都存储在 Cloudflare 的 D1 SQLite 数据库中。Cloudflare 的 D1 是一项第三方服务，有其自身的隐私政策，建议您查看
[**Cloudflare 通用隐私政策**](https://www.cloudflare.com/privacypolicy/) /
[**D1 数据安全**](https://developers.cloudflare.com/d1/reference/data-security/)
以了解他们如何处理您的数据。

### 3. 人机验证

本站使用 Cloudflare 的 Turnstile 服务防止垃圾邮件和滥用行为，当您提交订阅新闻简报的电子邮件地址时，Cloudflare 可能会收集以下信息:

- IP 地址
- 浏览器类型
- 访问时间

Cloudflare 的 Turnstile 是一项第三方服务，有其自身的隐私政策，建议您查看
[**Cloudflare 通用隐私政策**](https://www.cloudflare.com/privacypolicy/) /
[**Turnstile 隐私政策**](https://www.cloudflare.com/turnstile-privacy-policy/)
以了解他们如何处理您的数据。

## 数据安全

本站已尽力选择声誉良好、具备合理安全措施来保护您的个人数据免遭未经授权的访问、更改、披露或销毁。但请您理解，没有任何一种互联网传输或存储方式能够做到绝对安全。

如果您对数据安全有任何疑问可以随时与我联系。

如果您认为本站内容存在误解、不当或引发不适之处，欢迎随时与我联系沟通讨论。

## 政策变更

本站可能会不定时更新隐私政策，如果有任何变更将在本页面发布或修改政策并更新**最后更新**时间。修订后的政策自发布之日起生效，建议您定期查看本页面以了解最新内容。
