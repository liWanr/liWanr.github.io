## 说明

本站尊重您的隐私并致力于保护您的个人数据，所以最终呈现的是静态网站，不会收集任何您的数据，但依赖的外部服务可能会有收集数据，<u>**本站不对外部服务商的隐私政策负责**</u>，建议您查阅这些服务商的隐私政策。

## 收集的信息

### 1. 评论功能

本站使用 Giscus 服务提供评论功能，改服务本身不会收集任何信息。但其利用的 GitHub Discussions 可能会在您留言时收集以下信息：

- GitHub 账户信息
- IP 地址
- 浏览器信息
- 评论内容

Giscus 是一项第三方服务，有其自身的隐私政策，本站建议您查看
[**Giscus 隐私政策**](https://github.com/giscus/giscus/blob/main/PRIVACY-POLICY.md) /
[**GitHub 通用隐私政策**](https://docs.github.com/en/site-policy/privacy-policies/github-general-privacy-statement)，了解他们如何处理您的数据。

### 2. 订阅新闻简报

当您订阅本站的新闻简报时，会收集您的:

- 电子邮件地址
- 订阅日期和时间
- IP 地址（用于防止欺诈）

本站使用**站主自建的外部闭源订阅系统**管理新闻简报订阅，您的电子邮件地址仅用于发送简报通知，不会用于其他目的。您也可以随时点击任何简报邮件中的退订链接或直接与我联系以取消订阅。

其中所有数据都安全的存储在 Cloudflare 的 D1 SQLite 数据库中。Cloudflare 的 D1 是一项第三方服务，有其自身的隐私政策，本站建议您查看
[**Cloudflare 通用隐私政策**](https://www.cloudflare.com/privacypolicy/) /
[**D1 数据安全**](https://developers.cloudflare.com/d1/reference/data-security/)，了解他们如何处理您的数据。

### 3. 人机验证

本站使用 Cloudflare 的 Turnstile 服务防止垃圾邮件和滥用行为，当您在提交订阅新闻简报的电子邮件地址时，Cloudflare 可能会收集以下信息:

- IP 地址
- 浏览器类型
- 访问时间

Cloudflare 的 Turnstile 是一项第三方服务，有其自身的隐私政策，本站建议您查看
[**Cloudflare 通用隐私政策**](https://www.cloudflare.com/privacypolicy/) /
[**Turnstile 隐私政策**](https://www.cloudflare.com/turnstile-privacy-policy/)，了解他们如何处理您的数据。

## 数据安全

本站采取适当的技术和服务措施来保护您的个人数据免遭未经授权的访问、更改、披露或销毁。但是请您相信没有任何一种互联网传输方式是绝对安全的。

## 未成年人隐私

本站不面向 18 岁以下的未成年人，更不会故意收集未成年人的个人数据。如果您认为本站收集了未成年人的数据，请立即与我联系。

## 政策变更

本站可能会不时更新本隐私政策。如有任何变更将在本页面发布新政策并更新**最后更新**以通知您。
